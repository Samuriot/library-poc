# Lemieux Library Building Usage POC

## Local Dev
  1. Setup your local python venv
    ```python -m venv venv ```
  2. Install Dependencies
    ```pip install -r requirements.txt```
  3. Run locally
    ```fastapi dev```

## Endpoints
  - /health: health checking (expect 200OK)
  - /upload: uploading files (expect 201)
  - /stats: get parsed scripts (expect 201)
